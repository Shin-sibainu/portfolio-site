# Portfolio Website

ポートフォリオサイトのプロジェクトです。

## 概要

このプロジェクトは、Web開発者のポートフォリオサイトです。HTML、CSS、JavaScriptを使用して構築されています。
モダンなデザインとレスポンシブ対応で、様々なデバイスで快適に閲覧できます。

## 構成

- `index.html` - メインのHTMLファイル
- `styles.css` - スタイルシート
- `script.js` - JavaScriptファイル
- `hp/` - 追加のページやリソース

## 機能

- レスポンシブデザイン
- スムーズスクロール
- プロジェクト紹介
- お問い合わせフォーム
- モダンなUI/UXデザイン

## 使用方法

1. リポジトリをクローン
```bash
git clone https://github.com/Shin-sibainu/portfolio-site.git
```

2. ブラウザで `index.html` を開く

## 開発

- `master` ブランチ: 本番環境用
- `develop` ブランチ: 開発用
