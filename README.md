# Portfolio Website

ポートフォリオサイトのプロジェクトです。

## 概要

このプロジェクトは、Web開発者のポートフォリオサイトです。HTML、CSS、JavaScriptを使用して構築されています。

## 構成

- `index.html` - メインのHTMLファイル
- `styles.css` - スタイルシート
- `script.js` - JavaScriptファイル
- `hp/` - 追加のページやリソース

## 機能

- レスポンシブデザイン
- スムーズスクロール
- プロジェクト紹介
- お問い合わせフォーム

## 使用方法

1. リポジトリをクローン
```bash
git clone https://github.com/yourusername/my-portfolio.git
```

2. ブラウザで `index.html` を開く
